# Quiz 9d

## Assignment: map

Provide a function (or lambda expression) named 'twice' such that
```javascript copy
[1,2,3].map(twice) equals [2,4,6]
```

<RevealChildren>
    ```javascript copy showLineNumbers filename="Solution"
    const twice = x => x * 2;
    ```
</RevealChildren>

Your solution will be tested against:
```javascript copy
[1,2,3].map(twice).toString() === "2,4,6"
```
